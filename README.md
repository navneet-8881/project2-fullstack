# Full Stack Project

यह एक सरल फ़ुल-स्टैक प्रोजेक्ट है जिसमें React + Vite फ्रंटएंड और Express बैकएंड शामिल हैं।

## कैसे चलाएँ

1. प्रोजेक्ट रूट में जाएँ:
   ```bash
   cd "c:\Users\ng927\OneDrive\Desktop\full stack project\project2-fullstack"
   ```
2. डिपेंडेंसी इंस्टॉल करें:
   ```bash
   npm install
   ```
3. डिवेलपमेंट सर्वर चलाएँ:
   ```bash
   npm run dev
   ```

## क्या शामिल है

- `packages/backend` - Express API सर्वर
- `packages/frontend` - React + Vite UI

## API Endpoints

- `GET /api/message` - सर्वर संदेश देता है
- `POST /api/echo` - भेजा गया डेटा वापस लौटाता है
